# GfArch
Rust crate for handling Good-Feel's [GfArch files](https://swiftshine.github.io/doc/gfa.html).

## Capabilities
- [ ] Archive Creation
    - [ ] Archives with Byte Pair Encoding
    - [ ] Archives with LZ77
- [ ] Archive Extraction
    - [X] Archives with Byte Pair Encoding
    - [ ] Archives with LZ77
